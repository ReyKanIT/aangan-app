# Glossary

| Term | Meaning |
|------|---------|
| Aangan (आँगन) | "Courtyard" — the app name, represents shared family space |
| Dadi Test (दादी टेस्ट) | Design principle: UI must be usable by a grandmother |
| Haldi Gold | Primary brand color `#C8A84B`, used for headers and buttons |
| Mehndi Green | Accent color `#7A9A3A`, used for secondary actions |
| Level 1 / Direct | Immediate family (parents, siblings, spouse, children) |
| Level 2 / Close | Close relatives (grandparents, cousins, aunts, uncles) |
| Level 3 / Extended | Extended family and friends |
| RLS | Row Level Security — Supabase/Postgres access control per row |
| OTP | One-Time Password — phone-based login verification |
| FlutterFlow | No-code app builder used for Aangan's UI and logic |
